Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element allows you to seperate content from other content.
  • !-- -- allows you to add comments to your HTML code
  • The img element lets you add images to a webpage. It can also have attributes such as src (the source), alt (text used for screen readers),
  • the a (anchor) element can be used to link to another page. the attribute 'href' allows you to put the source, the attribute 'target' with the value '_blank' opens the link in a new tab.
  • ul and ol elements are unordered and ordered lists.

CSS

  • A margin indicates how much space we want around the outside of an element. You can also set specific margins with margin-'side'
  • A padding indicates how much space we want around the content inside an element.
  • font-family allows you to set the font.
  • font-size and font-style allows you to set the size and style of font.
  • color (must be american spelling!!!) allows you to set the colour of an object/text.
  • text-align allows you to set where the text is relative on the page. eg center, left, right.

GIT

  • git status: checks what branch we are currently on
  • git clone [url]: retrieves an entire repository from a hosted location via URL.
  • git branch: lists branches. *means active branch.
  • git pull: gets and merges any commits from the remote branch specified.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout [branch]: allows to to switch to a branch.
  • git add -A: adds all changed files to the staging area of Git.
  • git commit -m "message": commits what you have back to github
  • git push origin [branch]: pushes staged changes up to the githup repository. Ready for a pull request.
